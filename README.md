# Science Smash Skeleton

pygame 기반 2D 플랫폼 액션 / 대전 게임 기초 파일 뼈대입니다.

## 실행 방법

```bash
pip install -r requirements.txt
python main.py
```

## 현재 들어간 기능

- pygame 기본 실행 구조
- Game 루프
- Player 이동 / 점프 / 공격
- Boss 자동 움직임 뼈대
- Stage JSON 로딩
- Camera 시스템
- EventBus 이벤트 시스템
- Skill 시스템 뼈대
- 2P / Job 시스템 확장용 구조

## 조작

### Player 1
- A / D : 이동
- W : 점프
- F : 기본 공격
- G : 스킬 1

### 기타
- ESC : 종료
